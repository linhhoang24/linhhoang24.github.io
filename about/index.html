About

heyo is a responsive, minimalist, and pretty theme built for hugo, based on Alexander Bilz’s Anatole.

Features

  • Cover images
  • Pretty visualizations in the sidebar
  • Table of Contents in the sidebar
  • Callout shortcode
  • Badges shortcode
  • Dark theme
  • Support for custom CSS and JS
  • Multilingual support (english and brazilian portuguese, for now)
  • Open Graph support
  • Soon™

Quickstart

To try the exampleSite you just need to:

git clone https://github.com/LucasVadilho/heyo-hugo-theme heyo
cd heyo/exampleSite && hugo server --themesDir ../..

Live preview

You can check a live preview at heyo-theme.oioipio.com.
